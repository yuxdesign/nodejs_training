# nodejsの勉強用

```
$ git clone https://github.com/osonoi/nodejs_traonning.git
$ cd nodejs_training
```

## ウェブサーバーを起動
```
$ node app1.js
```
ブラウザーで　http://localhost:3000/　を確認
終わったらターミナルで　control + C でプログラム終了

## コメント変更
```

```
